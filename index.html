1. what is computer ?

A computer is an electronic device that processes, stores, and retrieves data based on instructions it's given.
It takes input, performs calculations and operations, and produces output.
Essentially, computers are machines that can store and process information, including words, pictures, and numbers.

2. what is RAM ?

RAM, which stands for Random Access Memory, is a type of computer memory that provides temporary storage for data and instructions that the computer is actively using.
It's like a computer's short-term memory, allowing the processor to quickly access information needed for running applications and performing tasks.
When the computer is turned off, the data in RAM is lost, unlike data stored on hard drives or SSDs.

3. what is MOUSE ?

A mouse is a pointing device used wh computers to control the cursor and interact with the graphical user interface.
It's a handheld device that typically has buttons for clicking and a scroll wheel for navigating.
Mice can be wired or wireless, connecting to the computer via a cable or radio waves, respectively.

4. what is KEY-BOARD ?

A keyboard is an input device used with computers and other electronic devices.
It allows users to enter text, numbers, and other commands by pressing buttons or keys.
Keyboards are essential for interacting with computers and are used for tasks like typing, navigating documents, and entering data.

5. what is CPU ?

A Central Processing Unit (CPU) is the primary component of a computer, often called the "brain," that executes instructions and performs calculations.
It is responsible for running the operating system, applications, and managing various computer operations.

6. what is PRINTER ?

A printer is a device that produces a physical, hard copy of digital information, typically text or images, onto paper.
It acts as an output device, transferring data from a computer or other digital source to a tangible medium.

7. what is SCANNER ?

A scanner is a device that captures images from documents, photographs, or other printed materials and converts them into digital files that can be stored, edited, and shared on a computer.
It's essentially a digital copier that transforms physical items into digital formats.
